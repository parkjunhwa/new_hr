경조신청안내

[경조구분]
  • 결혼, 출산, 회갑/칠순/팔순/구순, 사망, 탈상, 졸업 등의 경조발생 시 경조휴가 신청 가능
  • 각 항목별 경조금 및 휴가일수는 “경조안내” 클릭하여 확인
유의사항
  • 모든 경조휴가는 공휴일 포함한 기간임 (단, 배우자 자녀 출산은 공휴일 제외)
  • 결혼의 경우 해당일 하루 전부터 경조휴가 사용 가능하며 출산사용일의 경우 경조일로부터 경조휴가 사용 가능
  • 배우자 자녀출산의 경우 경조휴가 4번에 1회 사용(분할 3회) 가능 (단, 출산일로부터 12일 이내 사용)
  • 근무입사일 기준 근속기간 1년 미만의 경우 경조금의 50% 지급
  • 연말정산 기간 동안 e-HR → 개인신상정보 → 가족사항에서 가족 등록 시, 해당 구성원의 경조일 하루 전날 경조 안내 메일이 발송되므로 등록을 권장드립니다. (경조일은 등록사항 생년월일 기준)
[신청방법]
  • 경조 발생 시 담당자(인사전략팀 서명길 매니저 / Tel 02-2076-4649 / Email seomg@woongjin.com)에게 연락 바랍니다.
  • 직무상/휴가발생 경조 발생으로 담당자와 연락이 어려울 경우 인사전략팀 구성원에게 연락 주시면 안내 도와드리겠습니다.
  • 신청 버튼 클릭 → “경조휴가 신청” 팝업에서 상세 경조 항목 클릭 → 휴가 선택, 휴가일수 입력, 증빙서류 첨부 후 송신
  • 결혼이나 조의 등의 경우 확인자료를 미리 첨부하여 해당 정보입력
  • 경조사유 입력 후 경조휴가가 정상 신청되면 메일로 인사전략팀 담당자에게 제출하여 처리 가능
※ 경조 물품 배송 안내
  • 경조 물품 배송 시 지역에 따라 접수 마감 시간이 상이하므로 양해 부탁드립니다. (발송 후 3~4시간 소요 예상)
  • 서울, 수도권 당일 발송: 19~20시까지 접수 시 (이후 익일 발송)
  • 경상, 기타 소도시 당일 발송: 17~18시까지 접수 시 (이후 익일 발송)
경조사별 첨부서류
  • 본인결혼: 청첩장
  • 자녀결혼: 청첩장, 가족관계 증명원
